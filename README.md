# CodeChallenge CRUD-APP

Foobar is a Python library for dealing with word pluralization.

## Installation

Installation of client packages

```bash
cd client
npm install package.json
```
Installation of server packages
```bash
cd server
npm install package.json
```

## Usage

This run the client app
```node
cd client/
npm start
```
This run the server
```node
cd server/
npm start
```
